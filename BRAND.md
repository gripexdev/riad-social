# Riad Brand Guide

## Brand Name
**Riad** draws from the traditional Moroccan courtyard home: a calm, elegant space built for community, light, and conversation. The name positions the product as a refined, welcoming social space with cultural depth.

## Brand Personality
- Warm, inviting, and human
- Curated, intentional, and tasteful
- Confident without being loud
- Crafted, not generic

## Tagline Options
- Share your courtyard.
- Stories with warmth.
- Moments, gathered.

## Color System
Primary and secondary colors are inspired by Moroccan materials: terracotta clay, brass details, and jade tilework.

- **Terracotta (Primary / CTA):** `#C65A3B`
- **Brass (Secondary / Highlights):** `#D4A24C`
- **Jade (Accent / Success):** `#1F8A70`
- **Ink (Text):** `#1C1A17`
- **Muted (Subtext):** `#6A6158`
- **Line (Borders):** `#E4D8CB`
- **Cream (Surface):** `#F7F1E6`
- **Sand (Background):** `#EFE3D6`

### Gradient
- **Hero/CTA Gradient:** `linear-gradient(90deg, #C65A3B, #D4A24C)`

## Typography
- **Brand Typeface:** Marcellus (elegant serif with Moroccan warmth)
- **UI Typeface:** Sora (clean, modern, readable)

## Usage Guidelines
- Use **Terracotta** for primary actions and key highlights.
- Use **Brass** as a refined secondary accent and for gradient endpoints.
- Use **Jade** for positive states (success, active).
- Keep large areas of UI in **Cream/Sand** to preserve airiness.
- Pair strong imagery with soft surfaces and rounded corners.

## UI Tokens (Suggested)
- **Corner Radius:** 24px for cards, 999px for buttons
- **Shadow:** `0 18px 45px rgba(25, 18, 10, 0.12)`
- **Glow:** `0 16px 30px rgba(198, 90, 59, 0.28)`

## Example CSS Variables
```css
:root {
  --ink: #1C1A17;
  --muted: #6A6158;
  --line: #E4D8CB;
  --cream: #F7F1E6;
  --sand: #EFE3D6;
  --accent: #C65A3B;
  --accent-strong: #D4A24C;
  --jade: #1F8A70;
}
```
